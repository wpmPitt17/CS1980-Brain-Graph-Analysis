# CS1980-Brain-Graph-Analysis
CS 1980 Capstone - Brain Connectome Graph Analysis
Package of scripts for analysis of brain connectomes for use in classification of subjects based on correlation of brain connectome regions.
Analysis of different metrics of comparison to determine most effective ways to classify, identify, and visualize brain connectome data.

## Description
This repository is a part of the University of Pittsburgh CS1980 Capstone Course. The purpose of this project is the collect and analyze 
machine learning and data analysis applications towards brain connectome research. The package includes multiple isolated scripts for 
extracting and processing brain connectome data from fMRI data and machine learning applications with the processed data. These scripts
are meant to be resuable as a process of analyzing multiple facets to the brain connectome data from binary classification and labelling
of individual brain connectomes to identification of clustering utilizing K-NN and K-MEANS algorithms between connectomes.
